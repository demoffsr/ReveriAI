## 🎙 Record — Speak Your Dream

Wake up, tap record, speak. Reveri captures audio and transcribes in real-time with a **3-tier speech recognition engine** that never stops listening:

> **Tier 1** → `SpeechAnalyzer` (iOS 26+) — on-device, unlimited duration, 37+ locales<br>
> **Tier 2** → `SFSpeechRecognizer` — auto-restart on 1-min limit, silent reset detection<br>
> **Tier 3** → `WhisperKit` — planned offline fallback

**13 languages** · Russian · English · German · French · Spanish · Italian · Portuguese · Japanese · Korean · Chinese · Arabic · Turkish · Hindi

A **60fps animated waveform** visualizes your voice as you speak — rendered via `Canvas` + `TimelineView` with zero SwiftUI state overhead. Bars grow, scroll, and breathe with your voice.

<br>

## 🎨 Visualize — AI Paints Your Dream

Every dream can become a **unique artwork** generated by DALL·E:

1. **AI asks** 3 personalized visual questions about your dream
2. **You answer** — colors, mood, atmosphere, details
3. **AI enriches** answers into a 150-word art direction prompt
4. **gpt-image-1** generates a 1024×1024 high-quality image

The result is a one-of-a-kind illustration that lives with your dream forever.

## 📱 Live Activities

### 😴 Dream Reminder
Set bedtime → notification fires → **Live Activity stays on lock screen overnight** → wake up → tap Record or Write → deep link opens app → save dream → LA dismissed.

### 🎙 Recording Waveform
Active recording shows a **27-bar animated waveform** on the lock screen:
- Level sampling every 250ms with smoothing (attack/decay)
- Glass stop button + timer
- Gradient background with noise texture
- Dims to 40% on pause

### Core Principles

```
✦  Pure SwiftUI + Observation        No Combine, no UIKit
✦  @Observable ViewModels             @State private var viewModel pattern
✦  SwiftData persistence              Dream + DreamFolder with @Relationship
✦  Environment-driven theme           @Environment(\.theme) with auto day/night
✦  Observation isolation              ~43Hz audio updates in tiny wrapper views
✦  Unified glass API                  .reveriGlass(.circle) / .reveriGlass(.capsule)
✦  Xcconfig secrets                   Git-ignored → Info.plist → Bundle at runtime
```

## ☁️ Supabase Backend

### Edge Functions

| Function | AI Model | What it does |
|:---------|:---------|:-------------|
| `generate-dream-title` | GPT-4o-mini | 3-5 word title, movie-scene style |
| `generate-dream-interpretation` | GPT-4o-mini | Jungian analysis, 5 dimensions |
| `generate-dream-questions` | GPT-4o-mini | 3 personalized visual questions |
| `generate-dream-image` | gpt-image-1 | 1024×1024 dream artwork |
| `transcribe-audio` | Whisper | Audio → text transcription |
| `delete-dream-image` | — | Secure deletion with ownership check |
| `register-analytics` | — | Device registration + token |
| `track-event` | — | 50+ event types, batch support |
| `app-status` | — | Health dashboard API |

### Security Stack

```
🔐 Anonymous Auth       Persistent Keychain sessions, server-side validation
🚦 Rate Limiting        Per-user sliding windows (1min / 1hr / 24hr)
🛡️ Input Sanitization   wrapUserInput() prevents prompt injection
🔑 JWT Validation       Server-side getUser(token), ES256-compatible
🔒 SQL Security         REVOKE anon/authenticated, GRANT service_role only
🙈 Secrets              Git-ignored Secrets.xcconfig, never in source
```

## 🔧 Build

```bash
xcodebuild -scheme ReveriAI \
  -destination 'platform=iOS Simulator,name=iPhone 17 Pro' \
  build
```

<details>
<summary><b>Secrets setup</b></summary>

Create `Secrets.xcconfig` in the project root (git-ignored):

```ini
SLASH = /
SUPABASE_PROJECT_URL = https:$(SLASH)$(SLASH)<your-project>.supabase.co
SUPABASE_ANON_KEY = <your-anon-key>
```

The `SLASH` trick exists because `//` is parsed as a comment mid-line in xcconfig.
