# 🌙 Reveri — AI Dream Journal

> **Record dreams the moment you wake up. Let AI reveal what they mean.**

Reveri is a native iOS + watchOS dream journal that captures dreams via voice or text with real-time transcription, generates AI-powered interpretations through multiple psychological lenses, creates unique artwork for each dream, and presents it all in a beautiful dynamic interface that shifts between day and night.

**TestFlight** · [Status Dashboard](https://demoffsr.github.io/ReveriAI/status/) · [Privacy Policy](https://demoffsr.github.io/ReveriAI/privacy-policy.html)

## ✨ Features

### 🎙 Voice Recording with Live Transcription
Record your dream by speaking — the app transcribes in real-time using a **3-tier speech recognition engine**:

| Tier | Engine | Details |
|------|--------|---------|
| 1 | `SpeechAnalyzer` / `SpeechTranscriber` | iOS 26+ on-device, no time limit, 37+ locales |
| 2 | `SFSpeechRecognizer` | Fallback with auto-restart on 1-min limit, silent reset detection |
| 3 | WhisperKit | Planned — offline on-device |

Supports **13 languages**: Russian, English, German, French, Spanish, Italian, Portuguese, Japanese, Korean, Chinese, Arabic, Turkish, Hindi.

### 🧠 AI Dream Interpretation
After saving a dream, get a deep **Jungian analysis** generated by GPT-4o-mini across 5 dimensions:
- **Overview** — what the dream is about at its core
- **Archetypes** — Jungian archetypes present (Shadow, Anima/Animus, Self…)
- **Symbols** — recurring symbols and their psychological meaning
- **Emotional Context** — what emotions reveal about the subconscious
- **Key Symbols** — the most significant elements distilled

### 🎨 AI Dream Artwork
Each dream can have a **unique AI-generated image** (DALL·E / gpt-image-1):
1. AI asks 3 personalized visual questions about your dream
2. Your answers get enriched into a 150-200 word art direction prompt
3. A 1024×1024 high-quality image is generated and stored

### 📔 Dream Journal
- Chronological feed with **emotion tags** and search
- **7 emotions**: 😊 Joyful · 😍 In Love · 😌 Calm · 😕 Confused · 😰 Anxious · 😱 Scared · 😡 Angry
- **Dynamic emotion filter bar** with MRU ordering — selected emotion jumps to front
- **Folder system** — organize dreams into collections, drag & drop
- **Dream detail** — full text, audio playback, AI interpretation, generated image

### 🌊 Live Audio Waveform
Real-time animated waveform rendered at **60fps** via `Canvas` + `TimelineView`:
- Bars grow left-to-right during recording, respond to microphone level
- Playhead at 20% from left edge during playback for visible scrolling
- Skip/seek detection with automatic re-sync
- Zero SwiftUI state overhead — plain class buffer, isolated observation wrapper

### 🌗 Dynamic Day / Night Theme
The entire UI transforms based on time of day:

| | Day (5:00–21:00) | Night (21:00–5:00) |
|---|---|---|
| **Sky** | Dark starry header | Deep night sky |
| **Icon** | Golden sun with glow rings | Silver crescent moon |
| **Clouds** | Warm sand-colored, 3-layer parallax | Cool blue, 3-layer parallax |
| **Accents** | Orange/amber | Blue/indigo |

### 🔮 Liquid Glass Tab Bar
Custom floating tab bar with iOS 26 `glassEffect`:
- **Normal**: Record + Journal tabs with expand-on-tap
- **Recording**: Stop + Pause with 3D flip animation
- **Review**: Playback + Delete controls
- Emotion filter dynamically styles the Journal tab with the selected emotion's color and icon

```

├── App/                        # RootView, AppTab, entry point
├── Features/
│   ├── Record/                 # RecordView, RecordViewModel, TextModeView,
│   │                           # HowDidItFeelCard, SaveDreamButton
│   ├── Journal/                # JournalView, JournalViewModel, DreamCard,

│   │                           # filters, search, EmptyJournalView
│   └── Profile/                # ProfileView (locale, reminder settings)
├── Components/
│   ├── Header/                 # DreamHeader, CelestialIcon (sun/moon)
│   ├── Clouds/                 # 3-layer SVG cloud system (Back/Mid/Front)
│   ├── TabBar/                 # ReveriTabBar with glass effect
│   ├── AudioWaveform/          # Canvas + TimelineView waveform engine
│   ├── Toast/                  # ToastView (success/error)
│   └── EmotionPicker/          # EmotionBadge, EmotionPickerGrid
├── Services/
│   ├── AudioRecorder           # AVAudioEngine recording pipeline
│   ├── SpeechRecognitionService # 3-tier speech recognition
│   ├── DreamAIService          # AI title/image/interpretation orchestration
│   ├── SupabaseService         # Supabase client wrapper
│   ├── AuthService             # Anonymous persistent sessions
│   ├── LiveActivityManager     # Recording Live Activity lifecycle
│   ├── DreamReminderManager    # Bedtime reminder + LA
│   └── NotificationService     # Local notifications
├── Config/                     # SupabaseConfig (xcconfig → Info.plist → runtime)
├── Theme/                      # ThemeManager, ThemeColors, GlassButtonStyle
├── Models/                     # Dream, DreamFolder, DreamEmotion, SpeechLocale
└── Extensions/                 # Color+Hex, Date+Helpers

Reveri Watch App/               # watchOS companion (recording, sync, complications)
RecordingActivityWidget/        # Widget Extension (Live Activities)
```

### Key Principles

| Principle | Implementation |
|-----------|---------------|
| **Pure SwiftUI** | No Combine, no UIKit — Observation framework only |
| **@Observable ViewModels** | `@State private var viewModel = ViewModel()` pattern |
| **SwiftData** | `Dream` + `DreamFolder` models with `@Relationship` |
| **Environment theme** | `ThemeManager` via `@Environment(\.theme)` |
| **Observation isolation** | High-frequency updates (~43Hz audio) in small wrapper views |
| **Glass effects** | Unified `.reveriGlass(.circle/.capsule)` API |
| **Secrets** | Xcconfig → Info.plist → `Bundle.main.infoDictionary` at runtime |

### Recording State Machine

```
idle → recording → ┬─ has transcript? → text mode → idle (save)
                    └─ no transcript?  → reviewing → idle (save/delete)
```

## ☁️ Supabase Backend

### Edge Functions

| Function | Model | Purpose |
|----------|-------|---------|
| `generate-dream-title` | GPT-4o-mini | 3-5 word concrete title (movie-scene style) |
| `generate-dream-interpretation` | GPT-4o-mini | Jungian analysis across 5 dimensions |
| `generate-dream-questions` | GPT-4o-mini | 3 personalized visual questions |
| `generate-dream-image` | gpt-image-1 | 1024×1024 dream artwork from enriched prompt |
| `transcribe-audio` | Whisper | Audio → text transcription |
| `delete-dream-image` | — | Secure image deletion with ownership check |
| `register-analytics` | — | Device registration + token generation |
| `track-event` | — | Event logging (50+ event types, batch support) |
| `app-status` | — | Health dashboard API (public + admin) |

### Security

- **Anonymous auth** — persistent sessions via Keychain, server-side validation
- **Rate limiting** — per-user, per-function, sliding windows (1min / 1hr / 24hr)
- **Input sanitization** — `wrapUserInput()` prevents prompt injection
- **JWT validation** — `getUser(token)` server-side (bypasses gateway for ES256 compat)
- **SQL functions** — `REVOKE anon/authenticated`, `GRANT service_role` only
- **Secrets** — git-ignored `Secrets.xcconfig`, never in source

### Database

| Table | Purpose |
|-------|---------|
| `app_events` | All analytics events (50+ types) |
| `analytics_devices` | Registered devices with hashed tokens |
| `rate_limits` | Sliding window rate limit counters |
| `error_events_v` | VIEW — transforms `_failed` events for error dashboard |

## 🔧 Build

```bash
# iOS app
xcodebuild -scheme ReveriAI \
  -destination 'platform=iOS Simulator,name=iPhone 17 Pro' \
  build

# Requires: Xcode with iOS 26 SDK
# Simulators: iPhone 17 Pro, iPhone 17 Pro Max, iPhone Air
```

### Secrets Setup
Create `Secrets.xcconfig` in the project root (git-ignored):
```
SLASH = /
SUPABASE_PROJECT_URL = https:$(SLASH)$(SLASH)<your-project>.supabase.co
SUPABASE_ANON_KEY = <your-anon-key>
```
