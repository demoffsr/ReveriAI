# Reveri — Dream Journal & AI Interpreter

## Концепция

Reveri — iOS-приложение для мгновенной записи снов сразу после пробуждения с последующей AI-интерпретацией, аналитикой и статистикой. Главная ценность — скорость захвата сна, пока он свежий в памяти.

---

## Основные фичи

### 1. Быстрая запись сна
- **Голосовая запись** — юзер говорит, приложение записывает аудио + делает live-транскрипцию в реальном времени
- **Текстовая запись** — классический ввод текста
- Мультиязычность — запись на любом языке, AI переводит и интерпретирует

### 2. AI-интерпретация
- По нажатию «Interpret» на карточке сна, AI анализирует содержание
- Несколько линз интерпретации: юнгианская, фрейдистская, символическая, эмоциональная
- Выявление паттернов между снами: повторяющиеся символы, темы, связи

### 3. Журнал снов (Dream Journal)
- Хронологический список всех записанных снов
- Фильтрация по эмоциям, времени
- Поиск по содержанию снов

### 4. Статистика и аналитика
- Частота записи снов
- Распределение эмоций
- Повторяющиеся темы и символы
- Интеграция с HealthKit (Apple Watch): фазы сна, пульс, SpO2, корреляция физиологии с содержанием снов

### 5. Режим сна + Live Activity
- Юзер нажимает «Иду спать» → запускается Live Activity
- Live Activity висит на Lock Screen всю ночь
- Утром юзер видит карточку с кнопками «Record» и «Write» прямо на экране блокировки
- Live Activity не исчезает, пока юзер не запишет сон или не закроет вручную
- Обновляется каждые ~7 часов через Background Task, чтобы не истекал 8-часовой лимит Apple

### 6. Siri Shortcut
- «Записать сон» через Siri без открытия приложения (App Intents)

## Динамическая тема: День / Ночь

Приложение меняет визуальную тему в зависимости от времени суток:

- **05:00 — 21:00 (день):** тёмный хедер с звёздным небом, оранжевое солнце, тёплые песочные облака, оранжевые акценты
- **21:00 — 05:00 (ночь):** тёмный хедер с ночным небом, серебристая луна, голубые облака, голубые/холодные акценты

Тема применяется ко всем экранам консистентно.

## Экраны и флоу

### Таб-бар

Два таба, компактный дизайн:
- **Record** (левый) — иконка луны/микрофона, экран записи сна
- **Journal** (правый) — иконка журнала, список записанных снов

Поведение таб-бара:
- В сжатом состоянии — две иконки без лейблов
- При тапе на активный таб — раздвигающая анимация, показывает лейбл, затем сжимается обратно
- Во время голосовой записи таб-бар трансформируется в панель управления записью (Stop, Pause, Delete)

---

### Экран 1: Record (главный экран)

**Начальное состояние (Voice Mode — по умолчанию):**
- Хедер: «What did you dream about...?» (крупный жирный шрифт, «about...?» — оранжевым/голубым в зависимости от темы)
- Иконка солнца/луны справа вверху (с glow-эффектом и кольцами)
- Облака как разделитель между тёмным хедером и светлой нижней частью
- «Start Recording» слева (иконка микрофона + текст)
- «Text Mode» справа (иконка пера + текст)
- Таб-бар внизу

**Переход в Text Mode:**
- Поле ввода «Enter your dream...» заменяет «Start Recording»
- «Voice Mode» кнопка справа для переключения обратно
- Клавиатура выезжает автоматически
- При появлении клавиатуры облака и хедер поднимаются вверх (сжимаются), давая больше места для текста
- Когда текст введён, появляется «Save Dream» слева

---

### Флоу голосовой записи

**Шаг 1 — Начальный экран:**
- «Start Recording» + «Text Mode»
- Таб-бар в обычном состоянии

**Шаг 2 — Запись идёт:**
- Таймер: 00:00:01 — 00:05:00 (лимит 5 минут)
- Waveform (оранжевая аудиоволна) растёт в реальном времени
- Live Captions — транскрибированный текст появляется под waveform в реальном времени, последние слова выделены оранжевым (идёт распознавание)
- «Text Mode» скрывается
- Таб-бар трансформируется: «Stop» (красный, с лейблом) + «Pause» ⏸ (оранжевый, без лейбла)
- Предупреждение когда остаётся 30 секунд до лимита

**Шаг 3 — Пауза:**
- Waveform замирает
- Таб-бар: «Stop» + «Resume» ▶
- Юзер может продолжить запись

**Шаг 4 — Остановил запись (превью):**
- Таймер показывает финальную длительность: 00:00:00 — 00:05:00
- Waveform полная, статичная
- Финальный транскрибированный текст отображается под waveform
- «Save Dream» — кнопка справа вверху (основной CTA)
- Таб-бар: «Preview» ▶ (с лейблом) + 🗑 Delete (красная иконка корзины)
- Play/Pause воспроизведения — прямо на кнопке Preview в таб-баре
- Delete требует confirmation alert: «Discard this recording?»

**Шаг 5 — Сохранение:**
- Юзер нажимает «Save Dream»
- Toast-уведомление: «Dream saved ✓» (на 2 секунды)
- Экран возвращается к начальному состоянию
- Появляется плашка «How did it feel?» с крестиком ✕ — приглашение добавить эмоцию и детали
- Плашка висит пока юзер не тапнет на неё или не закроет крестиком

---

### Экран «How did it feel?» (после сохранения)

Появляется после сохранения сна (из плашки или по тапу):

**Выбор эмоции — 7 кастомных акварельных эмодзи:**

Порядок (от позитивного к негативному):
1. 🟢 **Joyful** — зелёный, улыбается
2. 💗 **In Love** — розовый, глаза-сердечки
3. 🟡 **Calm** — жёлтый, закрытые глаза, румянец, умиротворение
4. 🟣 **Confused** — фиолетовый, волнистый рот, непонимание
5. 🔵 **Anxious** — синий, со слезой, тревога
6. 🟠 **Scared** — оранжевый, широко открытые глаза и рот, страх
7. 🔴 **Angry** — красный, нахмуренные брови, злость

Стиль: кастомные акварельные иллюстрации (уникальный визуальный стиль Reveri, не стандартные эмодзи).

- Юзер тапает одну эмоцию → она выделяется
- Кнопка «Done» / автоматическое сохранение

---

### Экран 2: Journal (второй таб)

**Хедер:**
- Аватар юзера (слева) — тап открывает профиль/настройки
- Поле поиска «Search» (по центру)
- Заголовок «My Dreams»
- Сжатые эмодзи-фильтры справа от заголовка (маленькие акварельные эмоции в ряд)

**Эмодзи-фильтры:**
- В сжатом состоянии — маленькие иконки рядом с заголовком
- Горизонтальный свайп по ним → раскрываются, уезжая вправо, становятся крупнее
- Тап на эмоцию → фильтрует карточки по этой эмоции, эмодзи сжимаются обратно
- Выбранная эмоция остаётся визуально выделенной
- Повторный свайп + тап снимает фильтр
- Обрезанная последняя эмоция справа — визуальная подсказка что можно скроллить

**Сегментированный контрол:**
- Today | This week | This month
- «Today» выделен оранжевым по умолчанию

**Карточки снов:**
- Акварельная эмоция-иконка слева + название эмоции («Cheerful», «In Love», «Angry»...)
- «Interpret >» — оранжевая кнопка справа на каждой карточке
- Превью текста сна (2-3 строки, обрезается многоточием)
- Для голосовых записей: аудио-плеер с waveform и кнопкой ▶ (розовая/оранжевая)
- Дата и время записи (📅 Jun 12, 2025, 07:30am)
- Бейдж «✦ Translated» — если сон был записан на другом языке и переведён
- Белый фон карточки, скруглённые углы, лёгкая тень

**Пустое состояние (0 снов):**
- Большая акварельная эмоция-иконка (Joyful) по центру
- Заголовок: «Your dream journal is empty»
- Подзаголовок: «Record your first dream to start exploring your subconscious»

---

### Экран интерпретации (по тапу «Interpret >»)

- Полный текст сна наверху
- AI-интерпретация ниже
- Возможность выбрать линзу интерпретации (Юнг, Фрейд, символика, эмоции)
- Ключевые символы, выделенные из сна
- Связи с предыдущими снами (если есть паттерны)

---

### Экран настроек / профиль

Доступ через аватар на экране Journal:
- Настройки будильника / режима сна
- Интеграция с HealthKit (вкл/выкл)
- Язык интерпретации
- Уведомления
- Подписка / тарифный план

## Live Activity (Lock Screen)

**Запуск:** юзер нажимает «Иду спать» (кнопка в настройках или на главном экране)

**Вид на Lock Screen:**
- Иконка приложения (луна + горы) слева
- Заголовок: «Record Your Dream»
- Подзаголовок: «Don't forget to record dream»
- Две кнопки: «🎙 Record» + «✏️ Write»
- Кнопка ■ Stop (для отключения режима сна)

**Поведение:**
- Висит на Lock Screen пока юзер не запишет сон или не отключит вручную
- Обновляется через Background Task каждые ~7 часов (обход 8-часового лимита Apple)
- Тап на «Record» → открывает приложение сразу на экране голосовой записи (deep link)
- Тап на «Write» → открывает приложение в Text Mode
- На устройствах с Dynamic Island — компактный виджет сверху

## Технический стек

- **SwiftUI** — нативный iOS UI
- **SwiftData / CoreData** — локальное хранение снов
- **Speech Framework / Whisper API** — транскрипция голоса + live captions
- **Claude API / OpenAI API** — AI-интерпретация снов
- **HealthKit** — данные сна и физиологии с Apple Watch
- **WidgetKit + ActivityKit** — Live Activity на Lock Screen
- **App Intents** — Siri Shortcuts
- **AVFoundation** — запись и воспроизведение аудио
